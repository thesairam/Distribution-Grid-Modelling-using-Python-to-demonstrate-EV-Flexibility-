1. Install pandapower library with this command- pip install pandapower 

2. Please change the path for 'data.xlsx' file in the code to your local path before running. 

3. Run timeseries.py

4. Results are plotted and stored in results folder. 

If you would like to view only the results:

The folders Results1 and Results2 has already the results for 96 steps (24 hours). 
The folders Results1_short and Results2_short has already the results for 15 hours (60 steps) to focus 
on the flexibility part.

If you want to plot the results from these two folders, directly run the file plot.py.
